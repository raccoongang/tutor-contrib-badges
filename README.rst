Badges plugin for `Tutor <https://docs.tutor.edly.io>`__
########################################################

Badges plugin for Credentials service


Installation
************

.. code-block:: bash

    pip install git+https://github.com/raccoongang/tutor-contrib-badges

Usage
*****

.. code-block:: bash

    tutor plugins enable badges
